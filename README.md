# Pharmaceutical Batch Deviation Detection & CAPA System

**Author:** Harshitha Gudavalli  
**Degree:** M.S. Biomedical Engineering & Biotechnology  
**Regulatory Standard:** 21 CFR Part 211.192 — cGMP Production Record Review  
**Tools:** Python · pandas · NumPy · Matplotlib · openpyxl  

---

## Overview

An end-to-end GMP batch quality pipeline that processes 500+ simulated 
pharmaceutical batch records across 5 drug products — automatically 
detecting OOS results, classifying deviation severity, generating CAPA 
records, and producing a professional Excel audit report.

---

## What This Project Covers

- Automated OOS detection across assay %, pH, moisture, and particulates
- Deviation severity classification — Critical / Major / Minor
- Structured CAPA generation per 21 CFR Part 211.192
- 4-sheet Excel audit report with trend charts and color-coded findings
- Monthly OOS rate trend analysis across 500 batch records

---

## Regulatory Reference

- 21 CFR Part 211.192 — Production Record Review
- cGMP (current Good Manufacturing Practice)
- CAPA aligned with SOP-QC-007 Rev. 3
